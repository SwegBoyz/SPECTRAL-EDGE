# SPECTRAL-EDGE
**Energy-Efficient Distributed GNN for IoT Anomaly Detection**

## Project Overview
[cite_start]SPECTRAL-EDGE aims to develop a distributed Graph Neural Network (GNN) for real-time anomaly detection in IoT environments, optimized for resource-constrained edge devices (such as the ESP32)[cite: 2, 19]. 

This repository currently houses the environment setup and architectural foundations for Phases 1 and 2 of our project roadmap.

## Technical Environment Setup
[cite_start]The development environment has been successfully configured for local and edge simulation[cite: 33, 34, 35, 36, 38]:
* **Machine Learning:** Python 3.10+, PyTorch 2.x, PyTorch Geometric (PyG)
* **IoT Simulation:** NS-3 / GNS3
* **Edge Deployment:** Arduino IDE, ESP-IDF, TensorFlow Lite Micro
